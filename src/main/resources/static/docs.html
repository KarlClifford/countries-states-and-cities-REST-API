This is the Countries, States and Cities (CSC) API designed to the OpenAPI 3.1 specification. Here you can discover and test all of the endpoints included in this REST API!

city

Facilitates the creation, retrieval and modification of cities stored on the server.

Adds a new city to the server

Adds a new city (a name, the state it belongs to, the country that state is in, and its founding date. If the state and/or country do not already exist, then they will automatically be created. If a city of that name, in that state and country, already exists, then we will not add it.

Request Body schema: application/json
name
required
string

Name of the city

state
required
string

State the city belongs to

country
required
string

Country the state is located in

foundingDate
required
string <date>

Date in which the city was founded

Responses

http://localhost:8080/api/v1/city

Request samples

  • Payload
Content type
application/json
{
  • "name": "New York",
  • "state": "New York state",
  • "country": "United States of America",
  • "foundingDate": "02-09-1609"
}
